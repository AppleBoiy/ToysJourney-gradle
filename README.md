## How to run

- Clone this project

- Open terminal and run the following command

```bash
git clone git@github.com:AppleBoiy/ToysJourney-Gradle.git
```

- Compile java files

```bash
./gradlew build && ./gradlew jar
```

- Run the jar file

```bash
java -jar build/libs/ToysJourney-gradle-1.0-SNAPSHOT.jar
```
